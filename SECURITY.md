# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub Security Advisories:

https://github.com/farhanturu/ambatukam-go/security/advisories/new

Do NOT file public issues for security vulnerabilities.

We aim to respond within 7 days and provide a fix within 30 days for confirmed issues.

## Security Features

- Zero external dependencies — no supply chain risk
- All code reviewed and tested with race detector
- No known security vulnerabilities in current version
